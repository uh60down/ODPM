# ODPM

ODPM is a living product management manuscript about making decisions with enough shared understanding to move well.

The project treats product work as a discipline of clarity: framing problems, building decision readiness, chasing understanding, and keeping teams aligned around what matters.

## Structure

- `BOOK.md` is the current manuscript spine.
- `snapshots/` captures dated or numbered states of the thinking.
- `principles/` holds durable operating principles.
- `examples/` collects applied cases by domain.
- `glossary.md` defines recurring terms.

## Current Snapshot

Snapshot #002 introduces the PRD ontology: a way to describe product requirements as connected decisions, evidence, risks, constraints, and commitments.
